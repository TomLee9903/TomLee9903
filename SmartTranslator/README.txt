** SmartTranslator **

1. SmartTranslator.exe 실행

2. GUI의 파일 선택 버튼 클릭하여 번역 원하는 엑셀 파일 선택

3. GUI의 번역 타겟 중 원하는 타입의 번역 타겟을 선택

4. GUI의 번역 언어에서 원본(왼쪽) 및 번역본(오른쪽) 언어들을 선택

5. GUI의 번역 시작 버튼 눌러서 번역 시작

6. GUI의 조용한 모드를 선택하면 크롬 브라우저 오픈 없이 프로그램 사용 가능
   -> 조용한 모드 세팅 시, 다른 프로그램 1개에 한해 중복 사용 가능

* 파일 클리어 : 세팅되어 있는 파일의 데이터를 날리는 버튼

** 유의 사항 ** 

1. 번역 시작 후 크롬 브라우저를 한번 클릭해주세요. 원활한 번역을 위해 꼭 필요한 작업이니
   이 점 유의 부탁 드립니다.

2. 번역 언어의 경우 원본(왼쪽) 언어는 언어 감지 보다는 최대한 원본 언어로 설정 해주시는 게
   더 정확한 번역 결과를 기대할 수 있습니다.
   ex) 알리익스프레스의 옵션명 = 영어이면, 원본(왼쪽) = "영어" 로 선택
        징동닷컴의 상품 제목/상품 옵션명 = 중국어(간체)이면, 왼본(왼쪽) = "중국어(간체)" 선택

이용해 주셔서 감사합니다.
언제나 행복한 나날 보내세요~ 감사합니다.
