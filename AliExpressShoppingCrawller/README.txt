** AliExpress shopping crawller**

1. AliExpressShoppingCrawller실행

2. GUI의 상품명에 원하는 상품명 한글로 입력
   -> '상품검색 타입'을 상품명 수집 선택 시에만 입력하시면 됩니다.
   -> 그 외는 입력하지 않으셔도 됩니다.

3. GUI의 '검색 페이지 수' 란에 원하는 상품 페이지 수 입력
   - 수집 데이터 갯수 = 검색 페이지 수 * 60개

4. 상품검색 타입에서 원하는 방식의 상품 데이터 타입 선택
   -> 상품명 수집 외 타입 선택 시, 반드시 URL 오픈 후 해당 페이지로 이동한 후 '데이터 수집' 클릭해야
       정상적으로 데이터가 수집됩니다.

5. GUI의 URL 오픈 버튼을 클릭하여 알리익스프레스 웹 브라우저 오픈

6. 배대지, 언어, 통화를 수동으로 변경

7. 상품명 검색 버튼을 클릭하여 입력한 상품명 검색 창으로 이동
    -> 상품명 수집 외 타입 선택 시, 해당 버튼은 클릭하지 말아주세요.  

8. 무료 배송/별 4개 등 원하는 상품 정렬 방식 세팅

9. 데이터 수집 버튼을 눌러 데이터 수집 시작

10. 바탕화면의 '알리익스프레스_결과물' 폴더 내 수집 데이터 확인

** 유의 사항 ** 

1) 정확한 아이템 이름을 입력하지 않으면 검색이 제대로 되지 않을 수 있습니다.

2) 페이지 수를 입력하지 않으면 검색이 제대로 되지 않을 수 있습니다.

3) 'driver' 폴더가 프로그램과 같은 경로에 있어야 프로그램이 정상 동작 하오니 이 점 유의 바랍니다.

4) 엑셀 파일 양식 변경 시, 'driver' 폴더 내에 '결과양식.xlsx' 이름 그대로 양식만 변경하여 사용 부탁 드립니다.
    엑셀 파일 이름이 'sheet_reference.xlsx'이 아닐 시 프로그램이 정상 동작하지 않을 수 있으니 이 점 유의 바랍니다.
