** JingDong shopping crawller**

1. JingDongShoppingCrawller실행

2. GUI의 '로그인 정보' 란에 ID 및 PW를 입력한다

3. '초기설정' 버튼 클릭 후 뜨는 크롬 설정 화면에서 팝업 및 리디렉션 허용으로 변경

4. GUI의 '검색 방법' 란에 검색 방법을 선택한다
  4.1 상품명 검색 시 상품명과 검색 페이지 수 입력
  4.2 이미지 검색 시 네이버 수집기로 수집했던 엑셀 시트 선택

5. GUI의 '상품 타입' 란에 상품 수집 타입을 선택한다.
  5.1 상품명 - 상품명 란에 검색 원하는 상품명 입력 후 데이터 크롤링
  5.2 카테고리 - 징동닷컴 내 카테고리 상품의 URL 클릭 후 데이터 크롤링 (파일명은 자동으로 설정됨)

6. 로그인 버튼을 눌러 징동닷컴 URL 오픈 및 로그인

7. 검색 버튼을 눌러 데이터 수집 시작

8. 바탕화면의 '징동닷컴_결과물' 폴더 내 수집 데이터 확인

** 유의 사항 ** 

1) 정확한 아이템 이름을 입력하지 않으면 검색이 제대로 되지 않을 수 있습니다.

2) 페이지 수를 입력하지 않으면 검색이 제대로 되지 않을 수 있습니다.

3) 'driver' 폴더가 프로그램과 같은 경로에 있어야 프로그램이 정상 동작 하오니 이 점 유의 바랍니다.

4) 엑셀 파일 양식 변경 시, 'driver' 폴더 내에 '결과양식.xlsx' 이름 그대로 양식만 변경하여 사용 부탁 드립니다.
    엑셀 파일 이름이 '결과양식.xlsx'이 아닐 시 프로그램이 정상 동작하지 않을 수 있으니 이 점 유의 바랍니다.
