** 에덴밸리리조트 reservation AI Bot **

1. 압축파일 해제

2. EdenVally_GolfReservation.exe 실행

3. ID/Password 및 예약 희망 요일 선택

4. 각 코스 별 시작/종료 시간을 설정

5. 예약 타입을 선택한 후 Start 버튼 클릭
   1) 주말 자동 예약
        -. 매주 수요일 오전 9시 58분에 URL open 및 로그인 
        -. 9시 58분 30초에 예약 화면 띄움
        -. 10시 정각에 웹페이지 새로고침 후 예약 시작

   2) 평일 자동 예약
        -. 매주 월요일 오전 9시 58분에 URL open 및 로그인 
        -. 9시 58분 30초에 예약 화면 띄움
        -. 10시 정각에 웹페이지 새로고침 후 예약 시작

   3) 수동 예약
        -. '수동 예약' 선택 후 Start 버튼 누르는 즉시 URL open 및 로그인
        -. 예약 캘린더 띄움
        -. 예약 시작

   4) 예약 Re-try
        -. UI에 설정되어 있는 세팅으로 다시 재예약 진행
        -. 예약 캘린더 다시 띄워서 날짜 재선택
        -. 예약 시작

** 유의 사항 ** 

1) 원하시는 예약 타입 선택 후 필히 Start 버튼을 한 번만 눌러주세요. (웹페이지 오픈 전까지는 세팅을 마음대로 바꿀 수 있음)

2) '주말 자동 예약'과 '평일 자동 예약' 옆에 Cancel 버튼을 누르시면 각 타입별 자동 예약 설정을 해제합니다.
    자동 예약 설정 해제 시, 예약이 진행되지 않으니 이 점 참고 바랍니다.

3) 1)에서 Start 버튼을 2회 이상 클릭했을 시, 프로그램을 종료했다가 다시 실행하시기 바랍니다. 

4) time table 변경 시, 'driver' folder 내 에덴밸리리조트+타임테이블.xlsx를 수정 하시기 바랍니다.
   -. format은 현재 폴더에 있는 그대로 수정해야 프로그램 오류 없이 동작
   -. time table에 있는 시간대와 에덴밸리리조트 웹에 있는 시간대가 일치하는지 확인 후 사용 바랍니다.
   -. time table이 일치하지 않는 경우, 예약이 정상적으로 진행되지 않을 수 있으니 참고 바랍니다.

5) EdenVally_GolfReservation.exe와 driver folder가 같은 경로 내에 있어야 프로그램이 실행되오니 이 점 참고 바랍니다.

6) 모든 예약은 프로그램을 실행하는 현재 기준의 다음주이오니 참고 바랍니다.
   -. ex) 2022년 4월 10일 일요일에 프로그램을 실행하여 예약을 진행하면, 4/18 월 ~ 4/24 일까지가 타겟 날짜가 됨
