** 해운대CC reservation AI Bot **

<사용 방법>

1. 압축파일 해제

2. 'driver' folder 내 tesseract-ocr-w64-setup-v5.0.1.20220118.exe 실행 및 설치(default setting 그대로 설치)

3. HaeundaeCC_GolfReservation.exe 실행

4. ID/Password입력

5. 각 코스 별 원하는 예약 타임의 range 설정 (시작/종료 시간 선택)

6. 캘린더에서 원하는 예약 날짜 선택

7. 예약 타입 버튼 4개 중 원하는 버튼을 클릭 후 Start 버튼 실행

   7-1. 주말 자동 예약
	-> 매주 월요일 13시 58분에 해운대CC 웹사이트 오픈 및 자동 로그인
	-> 13시 59분 30초에 실시간 예약 페이지 이동
	-> 13시 59분 59초에 웹사이트 새로고침
	-> 14시부터 각 코스별 설정된 예약 타임 range 내 시간 예약 시작

	** '주말 자동 예약' 설정 후 Start를 누르고 프로그램을 계속 켜두시면, 예약이 매주 월요일 상기 시간대에 진행되오니 참고 바랍니다.
	** 예약 과정 시작 전에는 코스의 시간을 언제든지 변경하실 수 있으니 참고 바랍니다.
	** 프로그램을 종료하면 더이상 예약이 진행되지 않습니다.
	** 만약 세팅 변경이 불안하시면, '주말 자동 예약' 옆 'Cancel' 버튼을 클릭하여 예약 설정을 해제하실 수 있습니다.
	** Start 버튼은 필히 한번만 눌러주세요. 만약 두번 이상 클릭하셨을 시 프로그램을 재실행 해주시길 바랍니다.

   7-2. 평일 자동 예약
	-> 매일 8시 58분에 해운대CC 웹사이트 오픈 및 자동 로그인
	-> 8시 59분 30초에 실시간 예약 페이지 이동
	-> 8시 59분 59초에 웹사이트 새로고침
	-> 9시부터 각 코스별 설정된 예약 타임 range 내 시간 예약 시작
	
	** '평일 자동 예약' 설정 후 Start를 누르고 프로그램을 계속 켜두시면, 예약이 매일 상기 시간대에 진행되오니 참고 바랍니다.
	** 예약 과정 시작 전에는 코스의 시간을 언제든지 변경하실 수 있으니 참고 바랍니다.
	** 프로그램을 종료하면 더이상 예약이 진행되지 않습니다.
	** 만약 세팅 변경이 불안하시면, '주말 자동 예약' 옆 'Cancel' 버튼을 클릭하여 예약 설정을 해제하실 수 있습니다.
	** Start 버튼은 필히 한번만 눌러주세요. 만약 두번 이상 클릭하셨을 시 프로그램을 재실행 해주시길 바랍니다.

   7-3. 수동 예약
	-> 실시간 예약을 사용자가 원할 때 바로 실시 할 수 있는 기능
	-> 1~5번 설정 후 예약 타입에 '수동 예약' 버튼 클릭후 Start 버튼 클릭
	-> 설정된 시간 범위 및 날짜로 예약 바로 진행

   7.4. 예약 Re-try
	-> 예약에 실패했을 때, UI에 다른 시간 등을 설정하여 재예약을 할 수 있는 기능
	-> 예약 타입의 '예약 Re-try' 버튼 클릭 후 Start 버튼 클릭

<** 유의 사항 **>

1) 각 기능별 사용 방법을 참고 하시어 예약 방법에 혼선이 오지 않도록 주의 부탁 드립니다.

2) 설정된 시간의 최대 +/- 6분 차이의 시간까지 프로그램이 예약할 수 있도록 설정해 두었으니 참고 바랍니다.

3) time table 변경 시, 'driver' folder 내 '해운대CC+타임테이블.xlsx'를 수정 하시기 바랍니다.
   -. Time table format은 상기 파일대로 적용 되어야 프로그램 오류 없이 동작합니다.
   -. 파일명을 기존 '해운대CC+타임테이블.xlsx'과 동일하게 맞춰줘야 프로그램 오류 없이 동작합니다.
   -. time table에 있는 시간대와 해운대CC 웹에 있는 시간대가 일치하는지 확인 후 사용 바랍니다.
   -. time table이 일치하지 않는 경우, 예약이 정상적으로 진행되지 않을 수 있으니 참고 바랍니다.

4) HaeundaeCC_GolfReservation.exe와 driver folder가 같은 경로 내에 있어야 프로그램이 실행되오니 이 점 참고 바랍니다.

5) 캘린더에 원하시는 예약 날짜를 필히 선택 바랍니다. (UI에 선택된 날짜 check)
