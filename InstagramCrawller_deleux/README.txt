** Instagram AI Bot **

1. InstagramAIBot.exe 실행

2. Pop-up UI에 ID/PW 및 원하는 검색어를 입력한다
   -> 자동 댓글 입력을 원하시면 댓글탭 우측의 댓글 달기 check

3. 좋아요/팔로우 횟수에 원하시는 검색량을 입력
   -> 1, 23, 100 등의 자연수로 입력

4. 좌측 하단에 Start! 버튼을 눌러 인친님들의 피드를 감상하며 좋아요 팔로잉 및 댓글이 잘 달리는지 확인


** 유의 사항 ** 

1) Crawlling 갯수 최대 800개로 한정. 하루에 4~500개 추천.
   -> 봇 사용 간 계정 제한에 대한 책임을 지지 않습니다.

2) 간혹 web에서 log in이 실패하는 경우가 있습니다. 
   이 경우 프로그램을 재실행 하거나 컴퓨터를 재부팅 하면 해결 됩니다.

3) 피드 당 10초 ~ 100초의 랜덤한 delay를 설정하여, 봇탐지에 최대한 걸리지 않게 설정되어 있습니다.

4) 40개 단위로 crawlling 후 랜덤하게 5분 ~ 10분 간격으로 delay를 설정하였습니다.
   -> 인스타그램 홈페이지에서 프로그램을 봇으로 인식하지 않도록 하는 나름의 방어책입니다.최대한의 설정입니다.
   -> 본 delay가 인스타그램 홈페이지에서 봇으로 인식하지 않는다는 보장은 없습니다.
   -> 다만, 여러번의 시도 끝에 얻어낸 최적의 시간이오니 참고 부탁 드리겠습니다.

5) UI 우측 하단에 Re-login button을 체크하시면, 설정된 게시물 갯수만큼 검색 완료 후 재로그인하여 
   UI에 설정된 해시태그를 검색하여 다시 크롤링을 시작합니다. (원하시지 않으시면 체크박스를 해제하시고 사용 바랍니다.)
   -> 체크박스를 체크하시면 Re-login 주기를 입력할 수 있습니다. 
   -> Re-login 주기는 분 단위로 입력 바랍니다.(비어있을 시 10분으로 default)

6) 2단계 보안 설정되신 분들은 로그인 후, 웹사이트에 필히 수동으로 보안코드 입력 후 사용 부탁 드립니다.
   -> 5분까지 보안코드 입력 대기
   -> 5분이 넘어가면 프로그램 fail
