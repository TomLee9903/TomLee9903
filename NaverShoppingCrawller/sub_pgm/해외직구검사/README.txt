** NaverShoppingCrawller **

1. NaverShoppingCrawller.exe 실행

2. GUI의 '아이템 이름' 란에 원하는 아이템 이름을 입력한다

3. GUI의 '검색 페이지 수' 란에 데이터를 긁어오고 싶은 페이지 수를 입력한다
   -> 1, 23, 100 등의 자연수로 입력

4. GUI의 '금액 범위' 란에 설정하고 싶은 금액 범위를 입력한다
   -> 미입력 시 전체 범위의 상품 검색

5. 리뷰 란에서 리뷰 타입(리뷰 많은순/리뷰 좋은순)을 선택한다

6. 좌측 하단에 '검색' 버튼을 누르면, 검색한 아이템 및 페이지 수대로 데이터를 네이버 쇼핑에서 긁어와
   'YYYY-MM-DD_아이템이름_result.xlsx' 파일로 남는다.

** 유의 사항 ** 

1) 정확한 아이템 이름을 입력하지 않으면 검색이 제대로 되지 않을 수 있습니다.

2) 페이지 수를 입력하지 않으면 디폴트로 1개의 페이지만 크롤링 합니다.

3) 'driver' 폴더가 프로그램과 같은 경로에 있어야 프로그램이 정상 동작 하오니 이 점 유의 바랍니다.

4) 아이템 이름에 '/', ',' 등의 특수문자가 들어가면 크롤링 속도가 현저하게 떨어질 수 있으니 이 점 유의 바랍니다.
