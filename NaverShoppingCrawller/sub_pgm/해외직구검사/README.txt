** NaverShoppingCrawller **

1. NaverShoppingCrawller.exe 실행

2. GUI의 '아이템 이름' 란에 원하는 아이템 이름을 입력한다

3. GUI의 '검색 페이지 수' 란에 데이터를 긁어오고 싶은 페이지 수를 입력한다
   -> 1, 23, 100 등의 자연수로 입력

4. GUI의 '금액 범위' 란에 설정하고 싶은 금액 범위를 입력한다
   -> 미입력 시 전체 범위의 상품 검색

5. 리뷰 란에서 리뷰 타입(리뷰 많은순/리뷰 좋은순)을 선택한다

6. 결과물 폴더명 란에 원하는 결과물 폴더명을 입력한다
   (ex. 네이버 결과물, 네이버 엑셀 등, 만약 결과몰 폴더명 란이 비워져 있으면 default로 '네이버 결과물' 폴더 생성 후 결과물 저장함)

7. 좌측 하단에 '검색' 버튼을 누르면, 검색한 아이템 및 페이지 수대로 데이터를 네이버 쇼핑에서 긁어와
   'YYYY-MM-DD_아이템이름_result.xlsx' 파일로 남는다.

8. 바탕화면에 YYYYMMDD_네이버 폴더가 생성되어 각 아이템별 이미지가 저장된다.

9. 크롤링 종료 후 다른 세팅으로 크롤링 원할 시, 세팅 변경 후 '검색' 버튼 누르면 크롤링 재시작!


** 유의 사항 ** 

1) 정확한 아이템 이름을 입력하지 않으면 검색이 제대로 되지 않을 수 있습니다.

2) 페이지 수를 입력하지 않으면 디폴트로 1개의 페이지만 크롤링 합니다.

3) 'driver' 폴더가 프로그램과 같은 경로에 있어야 프로그램이 정상 동작 하오니 이 점 유의 바랍니다.

4) 엑셀 파일 양식 변경 시, 'driver' 폴더 내에 '결과양식.xlsx' 이름 그대로 양식만 변경하여 사용 부탁 드립니다.
    엑셀 파일 이름이 '결과양식.xlsx'이 아닐 시 프로그램이 정상 동작하지 않을 수 있으니 이 점 유의 바랍니다.
