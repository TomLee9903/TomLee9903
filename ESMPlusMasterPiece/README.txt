** ESMPlusMasterPiece **

1. MasterFile.xlsx 오픈 하여 업로드 원하는 ID/PW 기재

2. ESMPlusMasterPiece.exe 실행

3. GUI의 파일 당 데이터 개수 입력 (ex.100)

4. GUI의 파일 분할 버튼 클릭 후 분할 원하는 파일 선택
   (.xlsx 파일 포맷만 지원하오니, xls 파일은 꼭 xlsx 파일 확장자로 변경 부탁 드립니다.)

5. GUI의 업로드 시작 버튼 눌러서 업로드 시작

* 파일 클리어 : 세팅되어 있는 파일의 데이터를 날리는 버튼

** 유의 사항 ** 

1. 마우스 컨트롤을 이용하는 프로그램이오니, 프로그램 실행 후에는
   절대 마우스를 건드려서는 안됩니다. 그렇지 않을 경우 프로그램 정상 실행에 에러가 발생될 수 있습니다.

2. MasterFile.xlsx에 정확한 ID/PW를 입력해주세요.

이용해 주셔서 감사합니다.
